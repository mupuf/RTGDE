RTGDE
=====

A run-time generic decision-engine for power and performance management.

== Dependencies ==

Here is the list of dependencies to compile RTGDE:

 - cmake 2.6+
 - libpcap (optional)

== Compilation ==

Just run the following commands:

    $ git clone https://github.com/mupuf/RTGDE.git
    $ mkdir RTGDE/build
    $ cd RTGDE/build
    $ cmake ..
    $ make
